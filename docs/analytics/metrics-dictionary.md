# 通用指标

活跃与留存
- DAU/WAU/MAU：去重用户数；以 uid 或设备 id 去重
- 次日留存/7 日留存：次日/第 7 日回访用户 / 当日新增
- Session 时长：单次会话在线时长；异常短/长会话需剔除

转化与付费
- 新增付费率：新增用户中发生首付费的比例
- ARPU/ARPPU：总收入/DAU、总收入/付费用户数
- 付费留存：发生过付费的用户在 X 日的留存

性能体验
- 启动时长：应用从启动到首屏可用的时长（P50/P95）
- FPS：平均帧率；低帧率占比（< 30fps）
- 崩溃率：崩溃会话数 / 总会话数

# 维度与口径建议
- 统一时间粒度：分钟/小时/日；跨时区用 UTC 存储、展示按本地时区
- 维度白名单：platform、channel、country、version、game_type
- 高基数字段（uid、ip、trace_id）谨慎做聚合维度，避免卡方炸裂
